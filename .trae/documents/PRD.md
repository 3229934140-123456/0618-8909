## 1. 产品概述

共享充电宝运营管理系统是为充电宝运营方打造的一站式后台管理平台，涵盖点位管理、设备监控、借还计费、运维调度、数据分析和工单处理全流程。系统帮助运营方高效管理分散在各商场、餐厅、景区的充电宝设备，提升运营效率和用户满意度。

- 目标用户：充电宝运营方的后台管理人员、运维人员、客服人员
- 核心价值：实现设备全生命周期数字化管理，数据驱动扩量决策，降低运维成本

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 账号密码登录 | 全部功能：点位管理、设备管理、数据分析、工单处理、系统设置 |
| 运维人员 | 账号密码登录 | 查看补货/回收点位列表、扫码更新库存、设备状态查看 |
| 客服人员 | 账号密码登录 | 工单处理、用户投诉响应、借还记录查询 |

### 2.2 功能模块

1. **数据看板页**：核心运营指标总览、各点位借出率/收入/故障率、高流量点位识别
2. **点位管理页**：点位列表与筛选、新增/编辑点位、点位详情与设备关联
3. **设备管理页**：设备列表与状态监控、充电宝电量实时状态、低电量回收标记
4. **借还记录页**：借出/归还记录查询、自动计费明细、异常借还标记
5. **运维调度页**：补货点位列表、回收充电点位列表、扫码补货操作、运维任务状态
6. **工单系统页**：投诉工单列表、工单详情与处理、工单状态流转
7. **系统设置页**：计费规则配置、电量阈值设置、账户与权限管理

## 3. 核心流程

### 3.1 用户借还充电宝流程

用户在点位扫码借出充电宝，系统记录借出时间、点位和用户信息。用户使用完毕后在任意点位归还，系统自动计算费用并从绑定账户扣款。若超时未归还，系统发送提醒通知。

```mermaid
flowchart TD
    "用户扫码" --> "系统验证用户账户"
    "系统验证用户账户" --> "账户正常"
    "系统验证用户账户" --> "账户异常"
    "账户异常" --> "提示用户处理"
    "账户正常" --> "弹出充电宝"
    "弹出充电宝" --> "记录借出信息"
    "记录借出信息" --> "用户使用中"
    "用户使用中" --> "用户归还充电宝"
    "用户归还充电宝" --> "系统计算费用"
    "系统计算费用" --> "账户扣款"
    "账户扣款" --> "扣款成功"
    "账户扣款" --> "扣款失败"
    "扣款成功" --> "更新设备库存"
    "扣款失败" --> "标记欠费记录"
    "标记欠费记录" --> "发送催缴通知"
```

### 3.2 运维补货回收流程

设备端上报充电宝电量和位置数据，系统自动标记低于电量阈值的充电宝和库存不足的点位，生成运维任务。运维人员查看任务列表前往处理，完成后扫码确认更新系统库存。

```mermaid
flowchart TD
    "设备上报数据" --> "系统分析状态"
    "系统分析状态" --> "电量低于阈值"
    "系统分析状态" --> "库存不足"
    "电量低于阈值" --> "生成回收充电任务"
    "库存不足" --> "生成补货任务"
    "生成回收充电任务" --> "运维人员查看任务列表"
    "生成补货任务" --> "运维人员查看任务列表"
    "运维人员查看任务列表" --> "前往点位处理"
    "前往点位处理" --> "扫码确认完成"
    "扫码确认完成" --> "系统更新库存"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深青色 (#0F766E) + 琥珀色 (#F59E0B) 强调色，传达科技感与活力
- 辅助色：石板灰 (#1E293B) 作为深色背景，翡翠绿 (#10B981) 表示正常状态，珊瑚红 (#EF4444) 表示告警
- 按钮风格：圆角 8px，主要操作使用填充按钮，次要操作使用描边按钮
- 字体：思源黑体 / Noto Sans SC 作为界面字体，数字使用等宽字体 Tabular Nums
- 布局：左侧固定导航栏 + 顶部面包屑 + 右侧内容区的经典后台布局
- 图标：使用 Lucide 图标库，线条风格统一
- 数据可视化：使用平滑曲线和渐变填充，配色与主色调一致

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 数据看板 | 指标卡片 | 4列网格布局，每卡片含图标+数值+同比变化，微妙阴影和渐变背景 |
| 数据看板 | 趋势图表 | 面积折线图，渐变填充，悬停显示数据点详情 |
| 数据看板 | 点位排名 | 表格+条形图组合，前三名高亮，告警项红色标识 |
| 点位管理 | 点位列表 | 表格带筛选栏，类型标签使用彩色胶囊，状态圆点指示器 |
| 点位管理 | 新增点位 | 右侧抽屉式表单，分段填写，地图选点组件 |
| 设备管理 | 设备列表 | 卡片+列表双视图切换，状态标签色彩编码 |
| 设备管理 | 充电宝监控 | 柜机可视化示意图，槽位颜色表示电量等级 |
| 借还记录 | 记录列表 | 时间线+表格视图，费用高亮显示 |
| 运维调度 | 任务看板 | 看板式三列布局（待处理/进行中/已完成），拖拽更新状态 |
| 工单系统 | 工单列表 | 左侧列表+右侧详情的分栏布局 |
| 工单系统 | 工单详情 | 时间轴显示处理过程，快捷回复模板卡片 |
| 系统设置 | 计费规则 | 分段式表单，实时预览计费结果 |

### 4.3 响应式设计

- 桌面优先设计，最小支持 1280px 宽度
- 导航栏在窄屏下折叠为汉堡菜单
- 表格在窄屏下支持横向滚动
- 看板布局自适应列数
